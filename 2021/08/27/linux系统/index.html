linux系统

2021-08-27 Linux

Linux系统，具有运行稳定等很多优点，软件服务器大多部署在Linux系统上。

Linux 系统的优点

一、稳定与高效

Linux 内核的源代码是以标准规范的32 位(在64 位CPU上是64 位)的计算机来做的最佳化设计，可确保其系统的稳定性。正因为Linux 的稳定，才使得一些安装 Linux 的主机像Unix 机一样常年不关而不曾宕机。

在Linux中，大量的软件更新比任何其他操作系统中的更新都快得多。Linux中的更新可以轻松完成，而不会遇到任何重大问题。

磁盘快满时，Linux也会有效地执行所有任务。这提高了Linux的性能，因此Linux也提供了高性能。

Linux是多任务操作系统，Linux系统调度每一个进程是平等地访问处理器的，所以它能同时执行多个任务，而不会降低速度，例如下载大文件不会降低系统速度。

Linux在各种网络和工作站上提供高性能。它允许大量用户同时工作并有效地处理他们。

二、低配置要求

Linux可以轻松地从Web上安装，并且不需要任何先决条件，因为它可以在任何硬件上运行，甚至可以在最旧的系统上运行。

Linux帮助您将旧的和过时的计算机系统用作防火墙，路由器，备份服务器或文件服务器等。根据系统功能，可以使用许多发行版。您可以将Puppy Linux用于低端系统。

使用几乎所有的硬件，因为系统已经自带了驱动。

三、几乎免费

Linux可在网上免费下载和使用。您不需要购买许可证，因为Linux及其许多软件都随附于GNU通用公共许可证，这是Linux与Windows和其他操作系统相比面临的主要优势之一。

四、安全性

Linux系统自带防火墙、入侵检测、安全认证等工具，使得Linux系统是一个具有先天病毒免疫能力的操作系统。

Linux中的每个程序，无论是应用程序还是病毒，都需要密码形式的管理员授权。除非输入密码，否则不会执行病毒。Linux中不需要任何防病毒程序。

隐私权：Linux确保用户数据的私密性，因为它在使用发行版或软件时从不从用户那里收集大量数据

五、多用户共享

和Unix系统一样，Linux系统是真正的多用户多任务的操作系统。

Linux在各种网络和工作站上提供高性能。它允许大量用户同时工作并有效地处理他们。多个用户可以各自拥有和使用系统资源，即每个用户对自己的资源(例如：文件、设备)有特定的权限，互不影响，同时多个用户可以在同一时间以网络联机的方式使用计算机系统。

六、灵活性及兼容性

Linux运行或执行所有可能的文件格式，并且与大量文件格式兼容。

Linux可以轻松地从Web上安装，并且不需要任何先决条件，因为它可以在任何硬件上运行，甚至可以在最旧的系统上运行。

Linux提供了各种桌面环境以使其易于使用。在安装Linux时，您可以根据自己的意愿选择任何桌面环境，例如KDE（K桌面环境）或GNOME（GNU网络对象模型环境）。（比如你在linux安装了gnome和kde程序，那么切换时只需要输入：switchdesk gnome就能切换到gnome桌面环境了，输入switchdesk kde就能切换到kde桌面环境了；当然，现在windows下也能体验kde程序，不过首先需要安装kde installer【kde安装程序】，而且基于windows的kde程序现在好像还处于alpha【内部测试】阶段）

Linux提供了很大的灵活性，因为您只能安装必需的组件。无需安装完整或完整的套件。您还可以将Linux文件保留在多个分区下，因此，如果其中一个损坏，则不会造成重大损失。您只需要修复该特定分区，而不是其他操作系统无法修复的完整文件。

七、完备的软件支持及开发环境

安装了Linux系统后，用户常用的一些办公软件、图形处理工具、多媒体播放软件和网络工具等都已无需安装。而对于程序开发人员来说，Linux更是一个很好的操作平台，在Linux 的软件包中，包含了多种程序语言与开发工具，如gcc、cc、C++、Tcl/Tk、Perl、Fortran77 等。

Linux 内置了很丰富的免费网络服务器软件、数据库和网页的开发工具，如Apache、Sendmail、VSFtp、SSH、MySQL、PHP和JSP 等。近年来，越来越多的企业看到了Linux 的这些强大的功能，利用Linux 担任全方位的网络服务器。

八、大型用户社区

强大的用户社区能够随时随地帮你解决问题。
