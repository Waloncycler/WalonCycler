机器学习—数据预处理

2021-09-24 机器学习

利用 Sklearn 进行数据缺失值及异常值、标准化，归一化及正则化

缺失值

数据集中存在部分数据缺失的情况。

处理方法：

一、删除缺失数据点

二、数据补齐

  • 人工填写
  • 特殊值填充
  • 平均值（众数）填充
  • 热卡填充（就近补齐）
  • K最邻近法
  • 回归
  • 期望值最大化方法（EM）

三、不处理

直接进行数据挖掘，包括贝叶斯网络和人工神经网络等。

异常值

数据集中存在不合理的值， 即离群点。

标准化

将数据按比例缩放， 使之落入一个特定的区间。

  • 使用 sklearn.preprocessing.scale() 函数进行标准化处理
  • 使用 sklearn.preprocessing.StandardScaler 类，可以保存训练集中的参数（均值、方差）直接使用其对象转换测试集数据

归一化

把数据变成(0,1)之间的小数，量纲表达式转换成无量纲表达式。

  • 使用 sklearn.preprocessing.MinMaxScaler 类实现

正则化

将每个样本缩放到单位范数（样本单位范数为1），即对每个样本计算p-范数，然后每个元素除以改范数。
